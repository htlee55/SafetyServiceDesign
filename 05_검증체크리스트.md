# 검증 체크리스트

## 문제
- [ ] 실제 현장문제인가
- [ ] 사고 이전 위험 상태가 정의되었는가
- [ ] 사용자·운영자·책임자·지불자가 구분되었는가
- [ ] 줄이고 싶은 부담이 명확한가

## 폐루프
- [ ] 감지 이후 담당자가 지정되는가
- [ ] 행동 기준이 있는가
- [ ] 조치 완료와 위험 해소가 확인되는가
- [ ] 종료조건이 있는가
- [ ] 기록과 학습이 있는가

## 사람 중심
- [ ] 사람을 감시대상으로만 다루지 않는가
- [ ] 위험을 이해할 수 있는가
- [ ] 도움 요청이 가능한가
- [ ] 책임과 권한이 일치하는가

## 신뢰
- [ ] 설명 가능성
- [ ] 확인 가능성
- [ ] 통제감
- [ ] 프라이버시
- [ ] 공정성
- [ ] 이의제기

## 운영과 사업
- [ ] 기존 업무흐름에 들어가는가
- [ ] 알림 피로와 오류 대응이 있는가
- [ ] 지불자가 명확한가
- [ ] 반복가치가 있는가
- [ ] 성과를 증명할 수 있는가
- [ ] 유사 현장으로 확장 가능한가
