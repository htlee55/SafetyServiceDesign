# 안전안심서비스 핵심 프레임워크

## 안전
객관적 위험이 관리되는 상태:
위험 감소, 사고·near-miss 감소, 대응시간 단축, 조치 이행, 법규·감사 대응

## 안심
사람이 안전상태를 이해하고 믿는 경험:
설명, 확인, 통제감, 보호감, 프라이버시, 공정성, 결과 공유

## 고객이 구매하는 가치
위험 부담, 책임 부담, 운영 부담, 불안 부담, 비용 부담의 감소

## 지능 단계
| 단계 | 질문 |
|---|---|
| 감지 지능 | 무엇이 일어났는가 |
| 상황 지능 | 왜 위험한가 |
| 위험 전이 지능 | 어디로 악화되고 있는가 |
| 운영 지능 | 누가 무엇을 해야 하는가 |

## 폐루프
감지 → 이해 → 공동판단 → 행동 → 확인 → 학습

## 안전루프
위험 감지 → 판단 → 개입 → 위험 해소 확인

## 안심루프
상황 설명 → 확인 주체 공유 → 조치 완료 전달 → 재발방지 안내
