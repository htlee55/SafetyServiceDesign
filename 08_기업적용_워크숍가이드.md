# 기업 적용 워크숍

## 권장 명칭
AI-assisted 안전안심 서비스디자인 적용 워크숍

## 목적
자사 서비스를 안전안심 관점에서 재구성하고 현장검증 가능한 가설을 만든다.

## 준비물
- 현재 서비스
- 실패·정체 프로젝트
- 고객 요청 문제
- 신규 사업 아이디어
- 반복 민원

## 진행
1. 문제 선택
2. Discover
3. Define
4. Develop
5. Deliver
6. 검토

## 최종 산출물
- 문제정의문 1개
- HMW 3개
- 이해관계자 맵
- 위험 상태 전이
- 서비스 폐루프
- 안전루프·안심루프
- MVP
- 현장검증 질문 5개

## 핵심 문장
AI에게 답을 얻는 것이 아니라, 우리 서비스에 아직 답하지 못한 질문이 무엇인지 발견하는 것이 목적이다.
